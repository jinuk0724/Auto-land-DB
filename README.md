# 전국 부동산 인사이트

**전국 부동산 인사이트**는 공공데이터포털 서비스키를 사용해 전국 시·군·구 단위의 아파트 매매 및 전월세 실거래가를 조회하고, 실제 좌표가 확보된 거래만 지도에 표시하는 Windows 데스크톱 애플리케이션입니다. 현재 v2 배포본은 기존 실제 API 조회 흐름에 더해, 부산광역시 상권 점포이력 집계와 부산진구 유동인구 격자 표본을 앱 내 분석 데이터로 포함합니다.

## 실행 방법

앱은 연결된 Windows 컴퓨터의 다음 위치에 빌드되어 있습니다.

| 항목 | 경로 |
| --- | --- |
| 프로젝트 폴더 | `D:\AI\manus\real-estate-insight` |
| v2 부산 상권·유동인구 휴대용 실행 파일 | `D:\AI\manus\real-estate-insight\release\전국부동산인사이트-portable-v2-busan.exe` |
| v1 휴대용 실행 파일 | `D:\AI\manus\real-estate-insight\release\전국부동산인사이트-portable-v1.exe` |

실행하려면 `전국부동산인사이트-portable-v2-busan.exe`를 더블클릭하면 됩니다. 이 앱은 설치 없이 실행 가능한 **portable 단일 실행 파일**입니다.

## 주요 기능

| 기능 | 설명 |
| --- | --- |
| 통합 검색 | 지역, 거래 유형, 계약월, 키워드를 기준으로 부동산 데이터를 조회합니다. |
| 거래 유형 | 현재 아파트 매매와 아파트 전월세 조회를 지원하며, 상업업무용·경매·공매는 후속 확장 예정 항목으로 비활성 안내됩니다. |
| 지도 시각화 | Leaflet + OpenStreetMap 기반으로 실제 위도·경도가 확보된 거래만 표시하며, 좌표가 없는 거래는 결과 목록에만 표시합니다. |
| 전국 지역 선택 | 공식 법정동코드 전체자료를 기반으로 생성한 전국 시·군·구 선택 데이터를 제공합니다. |
| 부산 상권 v2 | 부산광역시 상권 점포이력 SHP 원문을 구·군 단위로 집계해 영업·폐업 점포 수, 주요 업종, 행정동 분포, 좌표 확인 상가 표본을 제공합니다. |
| 부산진구 유동인구 | 부산진구 S-서비스 유동인구분석 페이지의 격자 표본을 지도 마커와 유동인구 차트로 표시합니다. 부산진구 외 구·군은 현재 점포이력 기반 상권 분석을 우선 제공합니다. |
| CSV 내보내기 | 검색 결과를 CSV 파일로 저장할 수 있습니다. |
| 외부 링크 | 공공데이터포털과 법원경매정보 사이트로 바로 이동할 수 있습니다. |

## 공공데이터 API 키 설정

앱 좌측 패널의 **공공데이터포털 서비스키** 입력란에 사용자의 API 키를 입력한 뒤 **검색 및 분석 실행**을 누르면 실제 공공데이터 API를 조회합니다. 키가 없거나 API 호출이 실패하면 샘플 데이터로 대체하지 않고, 실패 원인과 조회 조건을 확인할 수 있도록 오류 메시지를 표시합니다.

사용 권장 데이터는 다음과 같습니다.

| 데이터 영역 | 데이터/API | 출처 |
| --- | --- | --- |
| 아파트 전월세 | 국토교통부_아파트 전월세 실거래가 자료 | 공공데이터포털 |
| 아파트 매매 | 국토교통부_아파트매매 실거래자료 | 공공데이터포털 |

## 개발 및 재빌드 방법

이 프로젝트는 Windows 컴퓨터에 Node.js를 전역 설치하지 않고, 프로젝트 내부의 portable Node.js를 사용합니다. PowerShell에서 다음 명령을 실행하면 다시 빌드할 수 있습니다.

```powershell
Set-Location 'D:\AI\manus\real-estate-insight'
$nodeDir=(Resolve-Path '.tools\node-v22.13.0-win-x64').Path
$env:PATH=$nodeDir + ';' + $env:PATH
& (Join-Path $nodeDir 'npm.cmd') run build
```

현재 v2 결과물은 Electron Builder의 Windows portable target으로 생성한 **단일 실행 파일**입니다. Windows 컴퓨터에 Node.js가 없는 경우에도, 개발 환경에서는 위 명령으로 소스 빌드를 검증한 뒤 별도 패키징 환경에서 portable 실행 파일을 생성할 수 있습니다.

## 제한 및 향후 확장

v2는 공공 API 키를 사용한 아파트 실거래 조회와 부산 중심 상권·유동인구 분석에 초점을 맞춥니다. 공공 API 응답에는 개별 단지의 위도·경도가 항상 포함되지 않으므로, 좌표가 확인되지 않은 거래는 지도에서 제외하고 목록·요약 통계에만 반영합니다. 부산 상권 데이터는 2023-12-31 기준 SHP 원문 집계이며, 유동인구는 부산진구 격자 표본부터 반영했습니다. 부산진구 외 구·군의 동일 수준 격자 유동인구는 원문 데이터 확인 후 확장할 수 있습니다. 법원경매정보는 공식 OpenAPI 확인이 어려워 기본적으로 외부 사이트 바로가기를 제공합니다. 민간 매물 사이트의 자동 크롤링은 각 사이트의 이용약관과 법적 위험을 고려해야 하므로 기본 기능에는 포함하지 않았습니다.

향후에는 주소 기반 지오코딩, 부산 전체 구·군 유동인구 원문 확대, 상가 CSV 로컬 인덱싱, SQLite 캐시, 행정동 경계 지도, 전국 상권 API 연동, 법원경매 유료 API 연동 등을 추가할 수 있습니다.

## 2026-05-31 빈 화면 문제 수정

초기 배포본에서 실행 창은 열리지만 화면이 비어 보이는 문제가 확인되었습니다. 원인은 Vite 빌드 결과의 JavaScript 및 CSS 참조 경로가 `/assets/...` 형태의 절대 경로로 생성되어, Electron의 `file://` 실행 환경에서 렌더러 리소스를 찾지 못한 것이었습니다. `vite.config.ts`에 `base: './'` 설정을 추가하여 `./assets/...` 상대 경로로 빌드되도록 수정했고, 별도 수정본 배포 폴더를 생성했습니다.

수정본 실행 파일은 다음 경로에 있습니다.

```text
D:\AI\manus\real-estate-insight\release\전국부동산인사이트-portable-fixed\전국부동산인사이트.exe
```

기존에 열려 있는 빈 화면 앱은 먼저 닫은 뒤, 위 수정본 실행 파일을 다시 실행해 주세요.

## API 키 기본값 설정

앱은 프로젝트 루트의 `.env.local` 파일에 `VITE_PUBLIC_DATA_SERVICE_KEY` 값을 저장하면 실행 화면의 공공데이터 서비스키 입력란에 기본값으로 자동 반영합니다. 이 파일은 `.gitignore`에 의해 GitHub에 업로드되지 않으므로, 공개 저장소에는 실제 인증키가 포함되지 않습니다.

