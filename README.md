# 전국 부동산 인사이트

**전국 부동산 인사이트**는 주거용·비주거용 부동산의 매매, 전세, 월세, 경매·공매 정보를 검색하고, 지도 기반으로 상권 및 유동인구 지표를 함께 확인할 수 있는 Windows 데스크톱 애플리케이션입니다. 초기 버전은 공공데이터포털 API 키가 있으면 실제 API를 조회하고, 키가 없으면 샘플 데이터로 전체 분석 화면을 검증할 수 있도록 설계되었습니다.

## 실행 방법

앱은 연결된 Windows 컴퓨터의 다음 위치에 빌드되어 있습니다.

| 항목 | 경로 |
| --- | --- |
| 프로젝트 폴더 | `D:\AI\manus\real-estate-insight` |
| 휴대용 실행 폴더 | `D:\AI\manus\real-estate-insight\release\전국부동산인사이트-portable` |
| 실행 파일 | `D:\AI\manus\real-estate-insight\release\전국부동산인사이트-portable\전국부동산인사이트.exe` |

실행하려면 `전국부동산인사이트-portable` 폴더 안의 `전국부동산인사이트.exe`를 더블클릭하면 됩니다. 이 앱은 설치형이 아니라 **휴대용 폴더 방식**이므로, 폴더 전체를 다른 위치로 복사해서 사용할 수 있습니다.

## 주요 기능

| 기능 | 설명 |
| --- | --- |
| 통합 검색 | 지역, 거래 유형, 계약월, 키워드를 기준으로 부동산 데이터를 조회합니다. |
| 거래 유형 | 전세·월세, 매매, 비주거용·상가 매매, 경매·공매를 선택할 수 있습니다. |
| 지도 시각화 | Leaflet + OpenStreetMap 기반으로 물건과 주변 상가 위치를 표시합니다. |
| 상권 분석 | 주변 업종 분포와 상권 매력도 지표를 차트로 보여줍니다. |
| 유동인구 분석 | 평일·주말 및 연령대 기반의 유동인구 추정 지표를 표시합니다. |
| CSV 내보내기 | 검색 결과를 CSV 파일로 저장할 수 있습니다. |
| 외부 링크 | 공공데이터포털과 법원경매정보 사이트로 바로 이동할 수 있습니다. |

## 공공데이터 API 키 설정

앱 좌측 패널의 **공공데이터포털 서비스키** 입력란에 사용자의 API 키를 입력한 뒤 **검색 및 분석 실행**을 누르면 실제 공공데이터 연동을 시도합니다. 키가 없거나 API 호출이 실패하면 앱은 자동으로 샘플 데이터 모드로 전환됩니다.

사용 권장 데이터는 다음과 같습니다.

| 데이터 영역 | 데이터/API | 출처 |
| --- | --- | --- |
| 아파트 전월세 | 국토교통부_아파트 전월세 실거래가 자료 | 공공데이터포털 |
| 비주거용 매매 | 국토교통부_상업업무용 부동산 매매 실거래가 자료 | 공공데이터포털 |
| 상권 정보 | 소상공인시장진흥공단_상가(상권)정보_API | 공공데이터포털 |
| 경매·공매 | 한국자산관리공사 온비드 부동산 물건목록 조회서비스 | 공공데이터포털 |

## 개발 및 재빌드 방법

이 프로젝트는 Windows 컴퓨터에 Node.js를 전역 설치하지 않고, 프로젝트 내부의 portable Node.js를 사용합니다. PowerShell에서 다음 명령을 실행하면 다시 빌드할 수 있습니다.

```powershell
Set-Location 'D:\AI\manus\real-estate-insight'
$nodeDir=(Resolve-Path '.tools\node-v22.13.0-win-x64').Path
$env:PATH=$nodeDir + ';' + $env:PATH
& (Join-Path $nodeDir 'npm.cmd') run build
```

Electron Builder의 일반 portable 패키징은 일부 Windows 권한 환경에서 심볼릭 링크 생성 오류가 발생할 수 있어, 현재 결과물은 Electron 런타임을 직접 복사한 **수동 휴대용 배포 폴더** 방식으로 생성했습니다.

## 제한 및 향후 확장

초기 버전은 공공 API 키를 사용한 실거래·공매 조회 구조와 샘플 기반 분석 UX를 제공합니다. 법원경매정보는 공식 OpenAPI 확인이 어려워 기본적으로 외부 사이트 바로가기를 제공합니다. 민간 매물 사이트의 자동 크롤링은 각 사이트의 이용약관과 법적 위험을 고려해야 하므로 기본 기능에는 포함하지 않았습니다.

향후에는 법정동 코드 전체 검색, 상가 CSV 로컬 인덱싱, SQLite 캐시, 행정동 경계 지도, 서울시 생활인구·상권분석서비스 연동, 법원경매 유료 API 연동 등을 추가할 수 있습니다.

## 2026-05-31 빈 화면 문제 수정

초기 배포본에서 실행 창은 열리지만 화면이 비어 보이는 문제가 확인되었습니다. 원인은 Vite 빌드 결과의 JavaScript 및 CSS 참조 경로가 `/assets/...` 형태의 절대 경로로 생성되어, Electron의 `file://` 실행 환경에서 렌더러 리소스를 찾지 못한 것이었습니다. `vite.config.ts`에 `base: './'` 설정을 추가하여 `./assets/...` 상대 경로로 빌드되도록 수정했고, 별도 수정본 배포 폴더를 생성했습니다.

수정본 실행 파일은 다음 경로에 있습니다.

```text
D:\AI\manus\real-estate-insight\release\전국부동산인사이트-portable-fixed\전국부동산인사이트.exe
```

기존에 열려 있는 빈 화면 앱은 먼저 닫은 뒤, 위 수정본 실행 파일을 다시 실행해 주세요.
